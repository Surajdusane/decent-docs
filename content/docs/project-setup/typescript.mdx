## Most Used Types in TS
```ts title="Types"
const a: string = 'hello';

const b: number = 1;

const c: boolean = true;

const d: null = null;

const e: undefined = undefined;

const f: void = undefined;

const g: unknown = null;

const h: any = 'hello';

const i: object = { name: 'John' };

array (e.g., string[] or Array<string>)

tuple (e.g., [string, number])

union (e.g., string | number)

type / interface (custom types)
```
